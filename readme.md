
immutables.github.io
====================

_Immutables_ annotation processor documentation site. (`immutables.org` redirects to `immutables.github.io`). AstroJS based, built and deployed with GitHub Actions.

The old Jekyll webside is in archive branches (`src-arhive-2025` and `master-archive-2025` (the former is the source of the site, the latter is pregenerated static website version)).
